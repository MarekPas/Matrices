# Matrices
Calculating matrices. It's one of the projects in JetBrains Academy.
 
## Description
The script allows:
- add matrices
- multiply matrix by a constant
- multiply matrices
- transpose matrix
- calculate determinant
- inverse matrix

All numbers of matrices must be entered with spaces.


## Technologies
* Python 3.7

## Setup
python Matrices.py
